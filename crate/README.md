# Water Guns vs Paper Planes
This crate contains the Rust to WASM components of Water Guns vs Paper Planes

### Compiling the crate
```
wasm-pack build
```

### Dependencies
* Rust (nightly)
* wasm-pack ^0.8.1

### Crates
* wasm-bindgen ^0.2
* js-sys ^0.3
* console_error_panic_hook ^0.1
* wee_alloc ^0.4
